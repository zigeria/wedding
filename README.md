# Свадебное приглашение — минималистичный редизайн (без градиентов)

Одностраничный сайт‑приглашение с чистой композицией и чёткой типографикой.


## Разделы
- Герой‑блок (дата и CTA)
- Где (адреса и ссылки на карты)
- План дня
- RSVP (localStorage)
- Опрос (localStorage)
- Галерея (рендерится из массива `window.GALLERY`)
- Как добраться

## Файлы
- `index.html` — обновлённая семантическая разметка и секции
- `styles.css` — полностью новые стили, адаптив, без градиентов
- `gallery.js` — список путей к изображениям (массив `window.GALLERY`)
- `script.js` — таймер, мобильное меню, рендер галереи, лайтбокс, RSVP и опрос
- `images/` — папка с фотографиями

## Данные и приватность
- Все данные RSVP и опроса сохраняются только в `localStorage` вашего браузера.
- Никаких сетевых запросов.

## Кастомизация
- Цвета/отступы/радиусы — в корневых переменных `styles.css`
- Тексты — в `index.html`
- Секции — можно скрывать/переставлять, редактируя `index.html`
